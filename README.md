
Описание проекта
В данном проекте выполнен анализ датасета Google Books с использованием Python, библиотеки pandas, numpy, matplotlib и seaborn, а также визуальная аналитика в Yandex DataLens.

Цель проекта — изучить структуру данных, выполнить их очистку, провести статистический анализ и визуализировать ключевые зависимости.

Предобработка данных
загрузка CSV-файла;
первичный обзор структуры данных;
обработка пропущенных значений;
удаление дубликатов;
анализ выбросов (Z-score).

Статистический анализ
описательная статистика (`describe`);
анализ уникальных значений;
корреляционный анализ числовых признаков.

Визуализация данных
гистограмма распределения количества страниц;
линейный график среднего рейтинга по годам;
boxplot для выявления выбросов;
heatmap корреляций.

Группировки и агрегации
анализ среднего рейтинга по категориям;
агрегации по языкам;
построение сводных таблиц (`pivot_table`).

Инструменты
Python 3
pandas, numpy
matplotlib, seaborn
Jupyter Notebook
Yandex DataLens

Содержимое репозитория
`Google_Books_Analysis_Jupyter_Commented.ipynb` — основной анализ данных
`google_books_dataset.csv` — исходный датасет
`README.md` — описание проекта
